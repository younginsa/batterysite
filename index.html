How it works

From input to insight in three steps

The interface stays simple while the solver does the heavy lifting on the server. You never touch the compute layer.

01

Pick & set conditions

Choose a cell from the library, then set C-rate, ambient temperature, and capacity through the web controls.

02

Run on the server

A containerized solver — your C, MATLAB, or Python code — receives the parameters, executes, and writes structured results back.

03

Read the results

Voltage curves and thermal contour plots render in the browser with key metrics. Export the raw data anytime.
